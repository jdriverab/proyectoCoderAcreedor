  • ¿Que es y quien puede ser un Acreedor?

  • Presta servicios a Sodimac para la mantencion y/o operacion de sucursales, ejemplo: servicio de inventarios, agencia de publicidad, etc.

  • Fabrica, vende y/o instala productos. graficas y muebles segun fichas tecnicas para sus exhibiciones y mantenciones.

  • Importa y/o comercializa insumos como EPP, POP, etc... para tiendas y CD.

  • Ser una empresa legalmente constituida y ubicada dentro del territorio nacional.

  • NOTA: La relacion con acreedores no considera la venta a traves de nuestros canales (en caso de querer vender productos ir al boton "proveedores").

  • ¿Que tipo de compras son y donde las requerimos?

  • Requerimos proveedores a lo largo de todo el pais para atender las necesidades de nuestras 75+ tiendas, CD y oficinas en chile.

  • Lo mas demandado actualmente son empresas para inventarios, inspectores tecnico de obras de prevencion y obras civiles, calculistas de redes internas y guardias.

  • Compra de servicios para la mantencion de nuestras tiendas: mano de obra de carpinteros para mantencion de las tiendas, guardias para seguridad, calculistas para exhibiciones, Itos, calculistas electricos, despachos.

  • Compras para fabricacion (segun nuestras fichas tecnicas), suministro y/o instalacion en tienda de exhibiciones, muebles, gancheras, roller, sillas, lockers, graficas y accesorios (metalicos, acrilicos, madera).

  • Compras para suministro de insumos tales como EPP, de venta POP, de merchandising, de tesoreria, agua para consumo, alarmas para productos (EAS), insumos covid, computadores para premios internos.
